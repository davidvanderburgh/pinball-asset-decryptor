# Pinball Asset Decryptor — Python dependencies
#
# The main application uses mostly the Python standard library.
# Pillow regenerates the app icon from source.
# zstandard powers the BOF May 2026+ PCK extractor/packer (Zstd
# decompression for the custom RSCC container BOF wraps fonts in).
# pycryptodome powers the BOF Dune write path: its PCK directory of
# absolute file offsets is AES-256-CFB encrypted, and we decrypt + re-encrypt
# it to repack edits of any size (Winchester's directory is plaintext).
# numpy powers the CGC Cactus Canyon decoders: the 185 MB usb.so audio
# de-encryption (byte-shuffle + XOR + 16-bit prefix-sum) and the cgc.so
# art de-obfuscation + RGB565->PNG render would be far too slow in pure Python.
# unicorn + capstone power the Stern Spike 2 audio engine: it drives the card's
# own game_real firmware in an ARM emulator to recover the per-sound codec
# keystream (decode every sound to WAV; re-encode replacements bit-exact).
# certifi supplies the CA bundle for the app's direct HTTPS calls (update
# check, AcoustID lookup): the frozen macOS builds link a Homebrew OpenSSL
# whose default CA path doesn't exist on non-Homebrew Macs, so without a
# bundled trust store every HTTPS request fails certificate verification.
Pillow>=10.0
certifi>=2024.2.2
zstandard>=0.22
pycryptodome>=3
numpy>=1.24
unicorn>=2.0
capstone>=5.0
